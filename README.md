# Основы JavaScript: Викторина
Данный репозиторий содержит исходный код для интерактивных заданий викторины.
## Основные темы:
* **Типы данных и операторы**: проверка базовых знаний синтаксиса.
* **Массивы и их методы**: использование `filter()` и `map()` для обработки данных.
* **Работа с DOM-деревом**: поиск элементов и обработка событий.
* **Логика и отладка**: поиск ошибок в циклах и условиях.
